# AstroBot Landing Page

A modern, responsive landing page for AstroBot – an automated trading bot that works on Binance, BingX, and soon Bitso.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern UI with gradient accents and smooth animations
- Sections: Hero, Features, Supported Exchanges, How It Works, Testimonials, Pricing, CTA
- Interactive elements (hover effects, counters, mobile menu)
- SEO‑friendly markup and semantic HTML
- Custom CSS with CSS variables for easy theming
- Vanilla JavaScript for lightweight interactivity

## File Structure

```
astrobot-landing/
├── index.html          # Main HTML document
├── style.css           # All styles
├── script.js           # Interactive functionality
├── favicon.ico         # Placeholder favicon
└── README.md           # This file
```

## Colors & Typography

- Primary: `#6c63ff` (purple‑blue)
- Secondary: `#00d4ff` (cyan)
- Dark background: `#0f0f1a` and `#1a1a2e`
- Light text: `#f8f9fa`
- Fonts: **Poppins** (body), **Orbitron** (headings)

## How to Use

1. Open `index.html` in any modern browser.
2. To edit styles, modify `style.css`.
3. To change behavior, edit `script.js`.

## Deployment

This page is ready to be deployed on any static hosting (GitHub Pages, Netlify, Vercel, etc.).

### GitHub Pages

```bash
# Clone the repository (if applicable)
git clone <your-repo>
cd astrobot-landing

# Commit and push
git add .
git commit -m "Add AstroBot landing page"
git push origin main

# Enable GitHub Pages in repo settings (branch: main, folder: /)
```

## Customization

- Replace the hero image with your own (update the Unsplash URL in `index.html`).
- Update exchange logos with official SVGs if available.
- Modify pricing plans and features in the Pricing section.
- Add real testimonials with photos.
